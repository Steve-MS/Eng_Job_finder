## What was delivered

### 28-Field Canonical Schema
Covers all required extraction targets: listing_id, source, source_url, title, employer, agency, location (normalized: "London, UK"), day_rate_min, day_rate_max, contract_type (permanent, contract, temp, freelance), duration_weeks, start_date, ir35_status (inside/outside/unknown), remote_policy (on-site, hybrid, remote), description_clean, posting_date, last_seen_at, extraction_confidence, extraction_method (API, regex, LLM), notes, dedup_group_id, canonical_flag, and technical metadata (created_at, updated_at, deleted_at).

### 3-Tier Extraction Strategy
1. **Fully Structured (API/HTML direct):** Reed JSON API fields map directly; title, employer, location from standardized HTML schema
2. **Regex-based:** Start date (regex: `\b(Monday|immediate|from\s+\d{1,2}\s+\w+|ASAP)\b`), duration (`\d+\s+(months|weeks|days)`), day rates (`£\d+k?\s*(?:per\s+day|day rate|pd)`), IR35 (`(inside|outside|umbrella|limited company)`), location normalization (postcode → region mapping)
3. **LLM fallback:** When regex fails or field is narrative, GPT-4o-mini extraction (cost ~$0.15 per 1M input tokens; use only for ambiguous title/description/IR35)

### Four Mandatory Hard Filters
1. **Contract Type Filter:** Pass if `contract_type in {contract, temp, freelance}`. Reject permanent. Precision ≥98%, Recall ≥92%.
2. **UK Geographic Filter:** Pass if location normalized to UK (postcode range, "UK", "remote UK"). Reject non-UK, vague "Europe". Precision ≥99%, Recall ≥95%.
3. **PM Role Filter:** Pass if title contains `(Project|Programme|Product|Portfolio) Manager`. Reject PMC, PA, Coordinator, PLC, Supervisor (false positives). Precision ≥92%, Recall ≥88%.
4. **Mechanical Engineering Domain Filter:** Pass if listing mentions `(mechanical|M&E|MEP|hvac|plant|equipment|manufacturing|automotive|aerospace|rail|energy)` in title or employer domain. Reject generic "engineering", "civil", "software". Precision ≥96%, Recall ≥90%.

### Jaro-Winkler Dedup Strategy
Compare pairs on:
- **Title similarity:** Jaro-Winkler >0.85 (e.g., "Senior PM – Rail" vs "Senior Project Manager – Railways")
- **Employer/Agency match:** Exact or Jaro-Winkler >0.90
- **Location overlap:** Same postcode region or within 10 miles
- **Rate band overlap:** day_rate_min/max ranges intersect or within ±£50/day
- **14-day posting proximity:** Both posted within 14 days of each other
- **Decision:** If ALL 5 conditions met, merge into dedup group; canonical record prioritizes most fields populated + highest source rank (Reed > Totaljobs > CWJobs > specialists) + widest rate range + most recent last_seen_at
